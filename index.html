Main website: here


Past projects

  • Election Contributions 2010-2016: another Kaggle kernel on FEC contributions
  • Drug References in Songs: quick Kaggle kernel on a lyrics dataset of top100 songs 1965-2015
  • Datafest 2018 presentation: work done for DataFest 2018, an exploration of a job listing platform’s data
  • ICO visualization: a Shiny app for visual exploration of some features of all ICOs as of 25.1.18 that had raised more than $10MM
  • Twitter personality: a Shiny app built for STAT231: Data Science, that visualizes the features of an existing dataset (n=3000) with matched personality scores and Twitter usage stats, and tries to predict the type of a new Twitter user via KNN-defined types using KNN-matching, technical appendix here
  • icow: relationships between certain country-level socio-economic variables and Date of Independence, ex-colonial ruler, etc.
  • shootings: visualizing 5-years of violent shootings in the USA
